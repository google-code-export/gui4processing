Creating your own color scheme for G4P

To create your own colour scheme for G4P you need 2 things

  1. a grahics drawing program that can create .png files e.g. MS Paint although something like Paint Shop Pro would be much bette.r
  2. an image file that contains all the colors for the schemes available. Right click here and select "save target as" to download the one that G4P uses.

This image file is 60 x 20 pixels and each row corresponds to a color scheme and each column represents one of the colours in the scheme.

Note that both row numbers and column numbers start from zero, so the top left pixel is (0,0).

Rows 0-6 are the 7 standard ones provided by G4P, I suggest that you leave these untouched and you add your schemes below.

 

When you have finished editing the file it must be saved as user_col_schema.png and placed in the data folder inside the sketch folder
