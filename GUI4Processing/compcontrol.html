Component Appearance

G4P has a number of methods to control the appearance of a component, the main ones are

 

Method: name(parameter type) Acceptable range of values for parameter Comments
setBorder(int) width >= 0 If 0 then the border will not be shown. At present I suggest that you only use values 0 and 1 larger values can have unwanted side effects on the appearance.
setOpaque(boolean) true or false Determines whether the background colour is used or not.
setAlpha(int) 0 to 255 incl Controls the level of transparency of the component (0 - fully transparent : 255 - fully opaque)
